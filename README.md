## Overview

This workshop provides a hands-on tutorial of [WEPPcloud](https://wepp.cloud) and [PATH](https://jackson-nakae.github.io/PATH-cost-effective/) (Pre/post-fire Assessment of Treatments for Hillslopes) --- two complementary tools for evaluating cost-effective post-wildfire hillslope treatments.

WEPPcloud is a web-based interface for watershed modeling built on the process-based Water Erosion Prediction Project (WEPP) model. It is currently used by Burned Area Emergency Response (BAER) teams and other land managers to simulate pre- and post-fire hydrology, erosion, and sediment delivery. PATH builds on these outputs, applying binary integer programming to identify the most effective combination of hillslope treatments needed to meet user-specified sediment and water-related thresholds under budget constraints.

The session is divided between an overview of the tools and hands-on group work where attendees use both tools with guidance from the presenters.
