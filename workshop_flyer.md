# WEPPcloud and PATH: A Cost-Effective Approach to Watershed Recovery

## After the Flames 2026 — Hands-On Workshop

**Monday, April 6, 1:00 PM – 5:00 PM**

---

## Overview

This 4-hour hands-on workshop provides an overview and tutorial of two complementary web-based tools for post-fire watershed assessment:

- **WEPPcloud** — A web-based interface for watershed-scale hydrologic and erosion modeling using the Water Erosion Prediction Project (WEPP) model
- **PATH** (Pre/post-fire Assessment of Treatments for Hillslopes) — An optimization tool that identifies cost-effective combinations of hillslope treatments based on user-defined monetary constraints and biophysical targets

The session will be divided between an overview of the tools by the presenters and hands-on group work where attendees will have the opportunity to use the tools with guidance.

## What You'll Learn

- How to use **WEPPcloud** to simulate pre- and post-fire hydrology, erosion, and sediment delivery across large landscapes
- How WEPPcloud automates GIS data acquisition, WEPP model runs, and statistical reporting through a web interface
- How to use the **Omni** feature to run multiple treatment scenarios simultaneously within a WEPPcloud project
- How **PATH** uses binary integer programming to identify the minimum-cost set of hillslope–treatment pairings needed to meet sediment and water-related thresholds
- How to evaluate treatment options (e.g., straw mulch at different tonnage per acre) under varying burn severities
